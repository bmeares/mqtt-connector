# `mqtt-connector` Meerschaum Plugin

The `mqtt-connector` plugin provides the `MQTTConnector`, which requires the following attributes:

The following optional attributes are accepted:

- `port`
- `username`
- `password`
- `keepalive`
- `transport`
- `clean_session`
